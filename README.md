# Challenge-Alura-Agente-OCI
Agente para el desafío Alura: Solución RAG para OCI (Operations Control Industrial). Automatiza la coordinación operativa petrolera mediante el análisis de procedimientos y reportes diarios. Optimiza la carga administrativa y logística con Gemini y LangChain, permitiendo a los supervisores enfocarse en decisiones críticas de campo.
