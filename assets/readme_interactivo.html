🤖 Agente Inteligente de Operaciones y Control Industrial (OCI)

Consola Interactiva RAG Avanzada con Embeddings Locales, Reranking Multilingüe e HITL en LangGraph

Este repositorio contiene el desarrollo del agente de Inteligencia Artificial para el Control de Operaciones Industriales (OCI). El sistema corre de manera unificada como un script de consola interactiva local (`RAG_OCI.py`) y está diseñado bajo el concepto de Human-in-the-loop (HITL), proporcionando un asistente operativo capaz de responder a consultas complejas utilizando la documentación corporativa, preservando el criterio del supervisor técnico.

📐 Arquitectura y Flujo de Trabajo

El agente utiliza un flujo de toma de decisiones gestionado mediante un Grafo de Estados en LangGraph que analiza riesgos operativos en tiempo real:

Flujo lógico de orquestación de estados de OCI
Inicio
Nodo: Triaje
Evaluador de Riesgo
Enrutamiento Condicional
RAG Pedir Info Supervisor
Auto-Resolver RAG Bajo Riesgo
Solicitar Detalles Incompleto
Alerta Supervisor (HITL) Crítico / HSE

🚀 Requisitos e Instalación Local

Ejecuta el asistente interactivo de consola desde tu terminal:

# Instalar dependencias
pip install -r requirements.txt

# Configurar variables de entorno .env con GEMINI_API_KEY

# Ejecutar consola interactiva
python RAG_OCI.py
